# Lama Agent — навыки для помощника лам

Монорепо с набором навыков (skills) для AI-агента, обслуживающего Московский Дацан «Гандан Чойпэлин».

## Структура

```
skills/
  raspisanie/   — генератор расписания хуралов (HTML + JPEG)
  ...           — будущие навыки
shared/         — общие утилиты
agent/          — ядро оркестратора
```

## Навыки

### raspisanie — Расписание хуралов

Генерирует красивое расписание из PDF-данных в форматах HTML и JPEG.

**Запуск:**
```bash
cd skills/raspisanie
node screenshot.js
```

**Что умеет:**
- Парсит расписание из PDF
- Генерирует HTML с фото Дацана, таблицей и подвалом
- Экспортирует в JPEG (deviceScaleFactor 2x)
- Поддерживает цветовую разметку особых хуралов

## Стек

- Node.js + Puppeteer (генерация JPEG)
- Vanilla HTML/CSS (шаблон)
